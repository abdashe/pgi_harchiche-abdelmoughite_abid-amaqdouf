# Framework de Pentest Mobile - Projet PFA

## 📱 Description Académique

Ce projet s'inscrit dans le cadre d'un Projet de Fin d'Année (PFA) en sécurité informatique.
Il vise à démontrer la compréhension des concepts de sécurité mobile à travers 
l'implémentation d'un framework simplifié d'analyse de vulnérabilités Android.

## 🎯 Objectifs Pédagogiques

- Comprendre l'architecture des applications Android (APK)
- Maîtriser les techniques de reverse engineering mobile
- Identifier les vulnérabilités OWASP Mobile Top 10
- Appliquer les bonnes pratiques de sécurisation
